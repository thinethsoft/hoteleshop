# Assets Directory for Capacitor Icons & Splash Screens

Place your source PNG files in this directory before running `npx capacitor-assets generate --android`:

1. `icon.png` (Required): Minimum 1024x1024 px PNG.
2. `splash.png` (Required): Minimum 2732x2732 px PNG (with your logo centered).

Once files are placed in this folder, run:
```bash
npm run generate-assets
```
